Detailed Description

C/C++ runtime APIs for supporting deep vision network model primitives and inferencing with TensorRT.

Available network primitives currently include:

  • imageNet for image recognition
  • detectNet for object detection + localization
  • segNet for segmentation

The different primitives each inherit from the shared tensorNet object.
